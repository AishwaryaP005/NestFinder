# 🏡 NestFinder
### Full-Stack Real Estate Platform

> A full-stack real estate web application designed to simplify property buying, selling, and management through an intuitive user experience and modern web technologies.

---

## 📌 Project Overview

**NestFinder** is a web-based real estate platform developed using **Java (JSP & Servlets)** following the **MVC architecture**. The application enables users to browse, list, and manage properties while facilitating seamless interaction between buyers and sellers.

The platform incorporates features such as **real-time chat using WebSockets**, an **EMI calculator powered by Java RMI**, appointment scheduling, property image uploads, and secure database management using **MySQL** and the **DAO design pattern**. Deployed on **Apache Tomcat**, NestFinder demonstrates the development of a scalable, full-stack web application with responsive UI and efficient backend processing.
